By Liezhou Zhong; 11 April 2024

Introduction

This nitrate table was extracted using the whole NitrateXplorer database, using global data and with the number of unique data sources for each included foods being more than 2. Therefore, not all the foods included in the database are presented in the table below. The full list of the included foods in the database please see the combined food table.

The NitrateXplore database contains a series of data fields related to samples, sampling location (city, country, region, continent), sampling time (month, season, year), analytic methods, etc. therefore, it is capable of filtering data and extracting nitrate content based on established protocols. In addition, you could combine several foods together to extract the desired nitrate content. For example, you can combine all data for "Lettuce", "Lettuce, Butterhead", "Lettuce, Iceberg", "Lettuce loose leaf" and "Lettuce, romaine" to extract a generic nitrate content for all lettuce cultivars.

Please CONTACT US to discuss the data extraction and calculation procedures that can work specifically for your purposes. We have developed a nitrate/nitrite calculator that can automatically and efficiently assign content values to food records while applying multiple protocols.
